Introducción al lenguaje

Sus siglas significan HyperText Markup Language (Lenguaje de Marcado de Hipertexto). No es un lenguaje de programación (como Python o Java), sino un lenguaje de marcado que sirve para definir la estructura y el contenido de una página web.

Etiquetas de formato

1.Hr:Significa salto de línea. Se usa cuando quieres bajar al siguiente renglón sin empezar un párrafo nuevo.
2.Br: también es un salto de línea, pero se usa dentro de un párrafo para separar líneas sin crear un nuevo párrafo.
